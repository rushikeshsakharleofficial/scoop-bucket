# scoop-bucket

Scoop bucket for [rushikeshsakharleofficial](https://github.com/rushikeshsakharleofficial) tools.

## Usage

```powershell
scoop bucket add rushikesh https://github.com/rushikeshsakharleofficial/scoop-bucket
scoop install code-outline-graph-go
```

## Tools

| App | Description |
|-----|-------------|
| `code-outline-graph-go` | Go code indexing and search MCP server |
